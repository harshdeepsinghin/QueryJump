# Release v1.0.0

**Release date:** 2026-04-11

## Summary

Initial stable release of QueryJump with production-ready packaging and version management.

## Features

- ✅ Single codebase for Chrome (MV3) and Firefox (MV2)
- ✅ Automated build and versioning via `build.sh` and `bump_version.sh`
- ✅ Manifest-based version sync across platforms
- ✅ Clean, reproducible builds

## Artifacts

- **QueryJump-Chrome-v1.0.0.zip** — Chrome extension package (MV3)
- **QueryJump-Firefox-v1.0.0.zip** — Firefox extension package (MV2)

## Release Process

Generated by running:
```bash
./build.sh  # Produces versioned archives
```

To release future versions:
```bash
./bump_version.sh <new_version>
```

This will automatically:
1. Update `manifest.json` with the new version
2. Build and package both browser extensions
3. Create a git commit and tag
4. Push to GitHub with release automation
